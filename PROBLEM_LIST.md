- O(log n) - Logarithmic
- O(n) - Linear
- O(n log n) - Linearithmic
- O(n¬≤) - Quadratic
- O(n¬≥) - Cubic
- O(2‚Åø) - Exponential

### Common Algorithm Complexities:
- Binary Search: O(log n)
- Linear Search: O(n)
- Bubble/Selection/Insertion Sort: O(n¬≤)
- Merge/Quick/Heap Sort: O(n log n)
- BFS/DFS: O(V + E)
- Dijkstra: O((V + E) log V)
- Dynamic Programming: Varies (O(n¬≤) common)
